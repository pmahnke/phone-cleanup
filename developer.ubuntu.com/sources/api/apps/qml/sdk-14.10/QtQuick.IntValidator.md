IntValidator
============

<span class="subtitle"></span>
Defines a validator for integer values More...

|                   |                      |
|-------------------|----------------------|
| Import Statement: | `import QtQuick 2.2` |

-   ****[bottom](index.html#bottom-prop)**** : int
-   ****[locale](index.html#locale-prop)**** : string
-   ****[top](index.html#top-prop)**** : int

<span id="details"></span>
Detailed Description
--------------------

The IntValidator type provides a validator for integer values.

If no [locale](index.html#locale-prop) is set IntValidator uses the default locale to interpret the number and will accept locale specific digits, group separators, and positive and negative signs. In addition, IntValidator is always guaranteed to accept a number formatted according to the "C" locale.
