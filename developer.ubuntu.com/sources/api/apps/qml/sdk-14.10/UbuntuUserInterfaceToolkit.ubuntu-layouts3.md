-   Layouts - Changing the order

Layouts - Changing the order
============================

<span class="subtitle"></span>
<span id="details"></span>
So far we have seen how to define the default layout, a conditional layout and how to place items into those.

As mentioned in the [ConditionalLayout](../Ubuntu.Layouts.ConditionalLayout/index.html) documentation, items hosted by the container are laid out based on the order they are hosted by the [ItemLayout](../Ubuntu.Layouts.ItemLayout/index.html).

So let's change the order the buttons are laid out so that we have blue, red and green order.

``` qml
ConditionalLayout {
    name: "row"
    when: layouts.width > units.gu(50)
    Row {
        anchors.fill: parent
        ItemLayout {
            item: "blue"
            width: parent.width / 3
            anchors {
                top: parent.top
                bottom: parent.bottom
            }
        }
        ItemLayout {
            item: "red"
            width: parent.width / 3
            anchors {
                top: parent.top
                bottom: parent.bottom
            }
        }
        ItemLayout {
            item: "green"
            width: parent.width / 3
            anchors {
                top: parent.top
                bottom: parent.bottom
            }
        }
    }
}
```

The layout after resizing the window width to exceed 50 GU will look as follows:

![](https://developer.ubuntu.com/static/devportal_uploaded/78240b28-974d-4ddf-8afb-ea6305f19d87-api/apps/qml/sdk-14.10/ubuntu-layouts3/images/layout2.png)

Note that when resizing the window width to &lt; 50 GU, you will get all your components back to their original (default) positions and anchors.
