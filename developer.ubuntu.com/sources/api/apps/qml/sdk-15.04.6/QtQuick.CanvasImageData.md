CanvasImageData
===============

<span id="details"></span>
Detailed Description
--------------------

The [CanvasImageData](index.html) object holds the image pixel data.

The [CanvasImageData](index.html) object has the actual dimensions of the data stored in this object and holds the one-dimensional array containing the data in RGBA order, as integers in the range 0 to 255.

**See also** [width](index.html#width-prop), [height](index.html#height-prop), [data](index.html#data-prop), [Context2D::createImageData()](../QtQuick.Context2D/index.html#createImageData-method), [Context2D::getImageData()](../QtQuick.Context2D/index.html#getImageData-method), and [Context2D::putImageData()](../QtQuick.Context2D/index.html#putImageData-method).
