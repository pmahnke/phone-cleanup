Detailed Description
--------------------

A [ContentStore](index.html) allows for the permanent storage of a transfered item.

See documentation for [ContentHub](../Ubuntu.Content.ContentHub/index.html) and [ContentScope](../Ubuntu.Content.ContentScope/index.html)
