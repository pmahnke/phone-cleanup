-   UI Components: Dial Control Example

UI Components: Dial Control Example
===================================

<span class="subtitle"></span>
<span id="details"></span>
This example shows how to create a dial-type control. It combines [Image](https://developer.ubuntu.com/api/apps/qml/sdk-15.04.6/QtQuick.imageelements/#image) elements with [Rotation](../QtQuick.Rotation/index.html) transforms and [SpringAnimation](../QtQuick.SpringAnimation/index.html) behaviors to produce an interactive speedometer-type dial.

![](https://developer.ubuntu.com/static/devportal_uploaded/a36d1cef-f63d-460f-bb0f-cef842d20a59-api/apps/qml/sdk-15.04.6/qtquick-customitems-dialcontrol-example/images/qml-dialcontrol-example.png)
