*Keyboard Focus in Qt Quick* combines various methods of handling keyboard focus. For more information, visit [Keyboard Focus in Qt Quick](../QtQuick.qtquick-input-focus.md).

<span id="running-the-example"></span>
Running the Example
-------------------

To run the example from Qt Creator, open the **Welcome** mode and select the example from **Examples**. For more information, visit Building and Running an Example.

Using the keyboard arrow keys, the focus shifts to the menus and the [GridView](https://developer.ubuntu.comapps/qml/sdk-15.04.3/QtQuick.draganddrop/#gridview).
