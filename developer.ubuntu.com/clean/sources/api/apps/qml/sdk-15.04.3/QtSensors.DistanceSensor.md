Detailed Description
--------------------

The [DistanceSensor](index.html) element reports the distance in cm from an object to the device.

This element wraps the QDistanceSensor class. Please see the documentation for QDistanceSensor for details.

**See also** [DistanceReading](../QtSensors.DistanceReading.md).
