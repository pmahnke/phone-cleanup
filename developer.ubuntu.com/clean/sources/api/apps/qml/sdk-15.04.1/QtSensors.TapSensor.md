Detailed Description
--------------------

The [TapSensor](index.html) element reports tap and double tap events along the X, Y and Z axes.

This element wraps the QTapSensor class. Please see the documentation for QTapSensor for details.

**See also** [TapReading](../QtSensors.TapReading.md).
