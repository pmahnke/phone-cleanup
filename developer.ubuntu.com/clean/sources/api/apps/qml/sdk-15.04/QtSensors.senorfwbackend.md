Sensorfw Backend
================

<span class="subtitle"></span>
<span id="details"></span>
The Sensorfw sensor backend requires the Sensorfw and sensor daemon. Sensorfw was originally used in MeeGo, and is available on : http://meego.gitorious.org/meego-middleware/sensorfw

To compile the sensorfw backend, you need sensorfw installed and run qmake with CONFIG+=sensorfw
