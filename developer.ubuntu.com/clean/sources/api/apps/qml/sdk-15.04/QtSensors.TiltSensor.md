Detailed Description
--------------------

The [TiltSensor](index.html) element reports tilt events along the X and Y axes.

This element wraps the QTiltSensor class. Please see the documentation for QTiltSensor for details.

**See also** [TiltReading](../QtSensors.TiltReading.md).
